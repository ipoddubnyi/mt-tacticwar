# TacticWar

Тактическая пошаговая стратегия.  
Аналог тактической части игры "Карибский кризис".
Язык программирования C#.

## Описание

Проект игры, который я начал делать в 2008 году на Visual Basic.
Сделал тогда карту, на которой можно перемещать танчики.  
  
В 2010 возобновил работу, но уже на C#. Улучшил карту, добавил миссии, подразделения, режим битвы, поиск кратчайшего пути.  
  
В 2019 году нашёл старые исходники, поразился, какую огромную работу тогда сделал, но не довёл до ума.
Решил начать с того, что переписать исходники, отделив логику от графики и т.п.
Залил проект в git, чтобы удобнее было отслеживать изменения и не делать резервные копии.
